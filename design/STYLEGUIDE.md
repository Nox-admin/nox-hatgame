# 🎨 Style Guide — iOS-приложение «Шляпа»

**Версия:** 1.0 · **Дата:** 10.03.2026  
**Платформа:** iOS · **Язык:** Русский (MVP)  
**Аудитория:** Россия, офлайн, все за одним телефоном

---

## 1. Визуальный стиль

**Концепция:** «Вечеринка у друзей» — тепло, игривость, ноч­ной уют. Не детское, не строгое. Как настольная игра, которую достают в пятницу вечером.

**Тон:** Дружелюбный, чуть хулиганистый. Короткие тексты, восклицания уместны, но без переборов.

---

### Основные
| Роль             | HEX       | Описание                          |
|------------------|-----------|-----------------------------------|
| Background Dark  | `#1A1040` | Фон — глубокий ночной фиолетовый  |

| Accent Gold      | `#F5C842` | Главный акцент — золото шляпы     |
| Accent Warm      | `#FF7A3D` | Вторичный — тёплый оранжевый      |
| Text Primary     | `#FFFFFF` | Основной текст                    |
| Text Secondary   | `#B8A8D9` | Подписи, хинты                    |

### Вспомогательные
| Роль             | HEX       | Описание                          |
|------------------|-----------|-----------------------------------|
| Success          | `#4ECB71` | Угадано ✅                         |
| Danger           | `#FF5252` | Пропуск ❌, таймер критический     |
| Surface Light    | `#352870` | Разделители, вторичные поверхности |
| Overlay          | `#00000080` | Затемнение под модалками          |

### Градиенты
- **Hero gradient:** `#1A1040` → `#3D2580` (вертикальный, для главных экранов)
- **Button primary:** `#F5C842` → `#E8A800` (горизонтальный)
- **Timer warning:** `#FF7A3D` → `#FF5252` (анимированный при < 10с)

## 4. Компоненты

### Кнопки

**Primary Button**
- Фон: градиент Gold
- Текст: `#1A1040` (тёмный на золоте)
- Высота: 56pt
- Corner radius: 16pt
- Shadow: `0 4pt 20pt rgba(245,200,66,0.4)`

**Secondary Button**
- Фон: `Surface Light` с border 1pt `#F5C842`
- Текст: `#F5C842`
- Те же размеры

**Destructive / Ghost Button**
- Фон: прозрачный
- Текст: `Text Secondary`

### Карточки
- Фон: `#2A1F5A`
- Corner radius: 20pt
- Padding: 20pt
- Shadow: `0 8pt 32pt rgba(0,0,0,0.3)`

### Игровые кнопки (Угадали / Пропустить)
- **Угадали:** круглая, 72pt, зелёный `#4ECB71`, иконка ✓
- **Пропустить:** круглая, 72pt, красный `#FF5252`, иконка ✕
- Haptic: Heavy при нажатии

### Input Field
- Фон: `#352870`
- Border: 2pt, по умолчанию `#4A3A80`, фокус `#F5C842`
- Corner radius: 12pt
- Placeholder: `Text Secondary`

## 5. Иконки и иллюстрации

- **Стиль иконок:** SF Symbols (нативные) + кастомные для игровых элементов
- **Ключевые иллюстрации:**
  - Шляпа (hero) — с лентой, объёмная, лёгкая 3D-стилизация
  - Звёздочки / конфетти — для экрана победителя
  - Карточки слов — иллюстрированные метки категорий
- **Формат:** SVG/PDF для масштабирования

---

## 6. Анимации и микровзаимодействия

| Момент                | Анимация                                          | Длительность |
|-----------------------|---------------------------------------------------|--------------|
| Запуск приложения     | Шляпа "вырастает" снизу, буквы вылетают          | 1.2с         |
| Смена экрана          | Slide + fade (iOS нативный)                       | 0.3с         |
| Новое слово           | Карточка "влетает" снизу с bounce                 | 0.4с         |
| Угадали               | Зелёная вспышка + частицы + haptic Heavy          | 0.6с         |
| Пропуск               | Красная вспышка + shake + haptic Medium           | 0.4с         |
| Таймер < 10с          | Цвет → красный, pulse-анимация, haptic каждую с  | —            |
| Конец игры            | Конфетти + fade-in победителя                     | 1.5с         |
| Передача телефона     | Blur экрана + prompt "Передай телефон"            | —            |

---

## 7. Spacing & Grid

- **Base unit:** 4pt
- **Экранные отступы:** 16pt (компактно) / 24pt (стандарт)
- **Между элементами:** 8 / 12 / 16 / 24pt
- **Safe Area:** строго соблюдается (notch, home indicator)
- **Touch targets:** минимум 44×44pt (HIG)

---

## 8. Тёмная тема

Приложение **только тёмное** — это выбор стиля, не системная настройка. Глубокий фон = ночная атмосфера вечеринки. Light mode не нужен для MVP.

---

## 9. Специфика для России

- Все тексты на русском, без англицизмов в UI
- Имена игроков могут быть кириллические — шрифт тестируется
- Слова в словарях — культурный контекст РФ
