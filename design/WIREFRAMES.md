# 📐 Wireframes — iOS-приложение «Шляпа»

**Версия:** 1.0 · **Дата:** 10.03.2026  
Описание экранов — структура, элементы, логика переходов.  
Всего экранов: **14** (добавлен выбор сложности словаря)

---

## Навигационная карта

```
Launch Screen
    └── Онбординг (3 шага) ──── пропустить ──┐
                                              ↓
                                      Главное меню
                                      ├── Правила
                                      ├── Настройки
                                      └── Новая игра
                                              ├── Настройки игры
                                              │       └── Сложность словаря ← NEW
                                              └── Ввод слов (per player)
                                                      └── Экран ожидания хода
                                                              └── Игровой экран
                                                                      └── Конец хода
                                                                              ├── → Игровой экран (след. ход)
                                                                              └── → Итоги раунда
                                                                                      ├── → Игровой экран (след. раунд)
                                                                                      └── → Финальные результаты
```

## Screen 04 · Настройки игры

```
┌─────────────────────────┐
│ ← Назад    Игра         │
├─────────────────────────┤
│                         │
│ Игроки                  │
│ ┌───────────────────┐   │
│ │ 👤 Игрок 1   [✎] │   │
│ │ 👤 Игрок 2   [✎] │   │
│ │ 👤 Игрок 3   [✎] │   │
│ │  + Добавить игрока│   │
│ └───────────────────┘   │
│                         │
│ Слов на игрока          │
│ ┌── [  -  ] [ 5 ] [  +  ] ──┐ │
│                         │
│ Время хода              │
│ ┌── [  -  ] [60с] [  +  ] ──┐ │
│                         │
│ Сложность словаря  [→]  │ ← переход на Screen 05
│ ┌───────────────────┐   │
│ │  Средняя      ▼  │   │
│ └───────────────────┘   │
│                         │
│  ┌─────────────────┐    │
│  │  Далее →        │    │
│  └─────────────────┘    │
└─────────────────────────┘
```

**Элементы:**
- Список игроков (свайп для удаления, tap для редактирования имени)
- Степпер: слов на игрока (1–15, default 5)
- Степпер: время хода (15с / 30с / 45с / 60с / 90с, default 60с)
- Сложность → переход на отдельный экран выбора или inline selector

## Screen 08 · Игровой экран (для объясняющего)

```
┌─────────────────────────┐
│  Маша   Осталось: 12 сл │ ← статус бар
│                         │
│   ╔═══════════════╗     │
│   ║               ║     │
│   ║   СЛОН        ║     │ ← ОГРОМНЫЙ текст, Display 48pt
│   ║               ║     │
│   ╚═══════════════╝     │
│                         │
│       ⏱ 00:45           │ ← круговой таймер (Timer 64pt)
│    [========    ]       │ ← progress arc вокруг числа
│                         │
│                         │
│   ╔═══╗         ╔═══╗  │
│   ║ ✕ ║         ║ ✓ ║  │
│   ╚═══╝         ╚═══╝  │
│   Пропуск      Угадали  │ ← большие круглые кнопки
│                         │
└─────────────────────────┘
```

**Критическое состояние (< 10с):**
- Таймер становится красным с pulse-анимацией
- Haptic feedback каждую секунду

**Конец времени:**
- Экран вспыхивает → автопереход на Конец хода

## Переходы между экранами

| Откуда → Куда                    | Анимация              |
|----------------------------------|-----------------------|
| Launch → Онбординг / Меню        | Fade                  |
| Онбординг слайды                 | Slide horizontal      |
| Меню → Настройки игры            | Push right (стандарт) |
| Настройки → Сложность            | Push right            |
| Настройки → Ввод слов            | Push right            |
| Ввод слов → Ожидание хода        | Push right            |
| Ожидание → Игровой               | Zoom in + fade        |
| Игровой → Конец хода             | Flash + Push up       |
| Конец хода → Ожидание            | Push right            |
| Конец хода → Итоги раунда        | Push up               |
| Итоги → Финал                    | Push up + confetti    |
| Любой → Меню (кнопка Назад)      | Pop left              |
