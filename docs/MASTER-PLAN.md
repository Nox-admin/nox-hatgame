# TASK-002 · Мастер-план: iOS-приложение "Шляпа"
_NOX Manager · 2026-03-10 · Статус: ожидает одобрения Jack_

---

## Что строим

Бесплатное нативное iOS-приложение для игры в шляпу.
Шляпа — словесная игра: игроки по очереди объясняют слова из общей колоды, остальные угадывают.

**Референс на Android:** [Шляпа — shlyapagame](https://play.google.com/store/apps/details?id=com.shlyapagame.shlyapagame) — 4.4★, 100k+ скачиваний, 13 000+ слов, тонкие игровые механики (режим «грабёж», жеребьёвка, сохранение игры).

## MVP — что входит в v1

- Создание игры с игроками (2–8 чел.)
- Встроенные наборы слов по уровням сложности (лёгкий / средний / сложный)
- Таймер с настройкой (30 / 60 / 90 / 120 сек)
- Кнопки «Угадали» / «Пропуск» + настройка штрафа
- Очерёдность ходов
- Режим «личная игра» (каждый сам за себя)
- Итоговый экран раунда + финальный экран победителя
- Сохранение и восстановление игры (прерывание)
- Онбординг (3 слайда)
- Русский язык

## После MVP (v2)

- Командный режим
- Режим «грабёж» (последнее слово может угадать любой)
- Жеребьёвка игроков
- Несколько раундов с теми же словами
- Создание своих словарей
- История игр

## Nice-to-have (v3+)

- Режимы объяснения (жесты, рисование, одно слово)
- iCloud синхронизация словарей
- iPad поддержка
- SharePlay для FaceTime
- Статистика игрока

**Экранов:** 13 (лонч, онбординг ×3, меню, создание, ввод слов, ожидание, игровой, конец хода, между раундами, финал, правила, настройки)

---

## Словари (задача для Analyst)

Три набора под разный уровень:
- **Лёгкий** (~150 слов) — простые, понятные всем
- **Средний** (~200 слов) — стандартная сложность
- **Сложный** (~150 слов) — редкие слова, термины, специфика

Итого: ~500 слов для MVP (vs 13k у Android-референса — цель на v2+)

## Модули системы

| Модуль | Содержание |
|---|---|
| Game Engine | Логика раундов, смена ходов, подсчёт очков |
| Deck Manager | Встроенные наборы по уровням, хранение |
| Player Manager | Игроки, очерёдность, жеребьёвка |
| UI / Screens | Все 13 экранов |
| Timer Service | Обратный отсчёт, звук/вибрация, фоновый режим |
| Storage | SwiftData: игры, колоды, настройки, сохранение/восстановление |
| Onboarding | Обучение новых пользователей |
